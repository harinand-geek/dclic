Test Flutter et Dart

Questions, Réponses et Explications Détaillées

Question 1
Quel langage est couramment utilisé pour interagir avec SQLite dans les applications Flutter ?
Réponse correcte : Dart
Dart est le langage natif de Flutter et est utilisé avec le package sqflite pour interagir avec SQLite.
Question 2
Comment définir une fonction "add" avec deux paramètres en Dart ?
Réponse correcte :
int add(int a, int b) => a + b;
Cette syntaxe utilise la fonction fléchée et le typage fort de Dart pour une définition concise.
Question 3
Qu'est-ce que SQLite ?
Réponse correcte : Un système de gestion de base de données relationnelles
SQLite est une base de données légère, serverless et auto-contenue, parfaite pour les applications mobiles.
Question 4
Quel widget est utilisé pour organiser des widgets horizontalement ?
Réponse correcte : Row
Le widget Row permet d'organiser ses widgets enfants horizontalement en ligne.
Question 5
Comment afficher une image depuis une URL en Flutter ?
Réponse correcte : Image.network
Image.network est le widget dédié pour charger et afficher des images depuis Internet.
Question 6
Quelle est la taille maximale d'une base de données SQLite ?
Réponse correcte : Il n'y a pas de limite théorique, mais la taille dépend du système de fichiers
La limite théorique est de 281 téraoctets, mais les contraintes pratiques dépendent du système de fichiers et du stockage disponible.
Question 7
Code Flutter pour le style d'un bouton
Réponse correcte :
final ButtonStyle stylebuton = ElevatedButton.styleFrom( backgroundColor: Colors.green, padding: const EdgeInsets.all(10), textStyle: const TextStyle(fontSize: 25), shape: const RoundedRectangleBorder( borderRadius: BorderRadius.circular(20)), elevation: 75.2 );
Configuration complète du style d'un bouton avec couleur, padding, style de texte et forme.
Question 8
Que représente une transaction dans SQLite ?
Réponse correcte : Un ensemble d'opérations SQL groupées, toutes ou aucune étant exécutée
Les transactions garantissent l'atomicité des opérations, assurant l'intégrité des données.
Question 9
Résultat du code avec opérateur ternaire
Réponse correcte : 10
L'expression (5 > 10) étant fausse, la valeur après le : (10) est retournée.
Question 10
Widget pour ajouter des marges intérieures
Réponse correcte : Padding
Le widget Padding est spécifiquement conçu pour ajouter des marges intérieures aux widgets.
Question 11
Étape essentielle lors de l'installation de Flutter
Réponse correcte : Configuration du chemin d'accès
La configuration du PATH est essentielle pour accéder aux commandes Flutter depuis n'importe quel répertoire.
Question 12
Différence entre StatelessWidget et StatefulWidget
Réponse correcte : StatelessWidget ne peut pas être mis à jour après sa création
Les StatelessWidget sont immuables, alors que les StatefulWidget peuvent être mis à jour dynamiquement.
Question 13
Utilisation de l'opérateur is! en Dart
Réponse correcte : Vérifier si un objet n'est pas du même type que le type spécifié
L'opérateur is! est utilisé pour la négation de la vérification de type.
Question 14
Widget de base pour afficher du texte
Réponse correcte : Text
Le widget Text est le widget fondamental pour l'affichage de texte dans Flutter.
Question 15
Fonctionnalité pour le développement rapide en Flutter
Réponse correcte : Hot Reload
Hot Reload permet de voir les changements de code instantanément sans redémarrer l'application.
Question 16
Définition d'une classe Personne en Dart
Réponse correcte :
class Personne { String nom; int age; Personne(this.nom, this.age); }
Utilisation de la syntaxe concise de Dart pour les constructeurs avec initialisation directe des propriétés.
Question 17
Comment spécifier la taille de la police pour un widget Text ?
Réponse correcte : En utilisant style avec TextStyle
Text( 'Hello World', style: TextStyle(fontSize: 20.0), )
TextStyle permet de définir les propriétés de style du texte, incluant la taille de la police.
Question 18
Comment aligner un widget au centre de son parent ?
Réponse correcte : En utilisant le widget Center
Center( child: Text('Texte centré'), )
Le widget Center est la solution la plus simple et directe pour centrer un widget dans Flutter.
Question 19
Principal avantage de Dart pour le développement d'applications
Réponse correcte : Unification du développement
Dart permet un développement unifié sur différentes plateformes avec un seul code base, simplifiant le développement multiplateforme.
Question 20
Comment gérer l'état d'une application Flutter ?
Réponse correcte : En utilisant setState() dans les StatefulWidget ou en utilisant des gestionnaires d'état comme Provider ou Bloc
Flutter offre plusieurs approches pour la gestion d'état, depuis la solution intégrée setState() jusqu'aux solutions plus avancées comme Provider ou Bloc pour les applications complexes.
